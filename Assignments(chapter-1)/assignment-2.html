Q.N4. Use of text formatting:

Text formatting in HTML refers to the way text is displayed on a web page. It is the process of applying various styles, colors, fonts, sizes, and other visual enhancements to text content within an HTML document.

Element Name Description
b used to bold the text written between it.
strong tells the browser that the text is important.
i used to make text italic.
em used to display content in italic.
mark used to highlight text
u used to underline text written between it.
sup displays the content slightly above the normal line
sub displays the content slightly below the normal line.
del used to display the deleted content.
ins displays the content which is added
big used to increase the font size by one conventional unit
small used to increase the font size by one conventional unit
