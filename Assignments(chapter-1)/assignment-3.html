HTML table :

An HTML table is a fundamental structure used for organizing and displaying data in rows and coluns on a web page.
key components :

Table:
- Defined using the 'table' tag, it serves as the container for all table elements.
Rows:
- Created with the 'tr' tag, each tr (table row) conatins one or more table data cells ('td') or table header cells('th').
Cells:
- 'td' (table data cell) and 'th' (table header cell) are used to define the individual data points within each row. 'th' is typically used to define headers for columns or rows.
Headers:
-Optionally defined using 'head', 'tbody', and 'tfoot' tags:
  • 'thead': Contains header rows in the table.
  • 'tbody': Contains the main content rows.
  • 'tfoot: Contains footer rows.
Attributes:
- Tables can have attributes like border, width, cellpadding, cellspacing, etc., to control spacing, borders, and overall appearance.

HTML Lists:

HTML lists are used to organize and structure content into ordered (numbered) and unordered (bulleted) lists. They are essential for presenting information in a clear and hierarchical manner on web pages. Here are the key types of lists in HTML:

Unordered lists
  • Created using the 'ul' tag.
  • Each list item is defined with 'li' tags
  • Displayed with bullet points by default, which can be customized using CSS.
    • ordered lists
  • Created using the 'ol' tag.
  • Each list item is defined with 'li' tags
  • Displays items with numbers or other specified markers (like letters or Roman numerals).
    • Description lists
  • Created using the 'dl' tag.
  • Consists of pairs of terms ('dt') and their descriptions ('dd').
  • Useful for glossaries or defining key terms.
